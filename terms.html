2. Service Description

PakPhantom is a multi-purpose Discord bot that provides:

  • 94 slash commands across 6 categories
  • 22 interactive games (casino, word, arcade, board, quiz, classic)
  • Fun interaction commands with GIFs
  • Marriage and relationship system
  • Virtual economy system with coins
  • Utility tools (reminders, notes, polls, etc.)
  • API-powered content (weather, animals, anime, movies)
  • Pakistani/Desi cultural content

All features are provided free of charge with no premium tiers or paywalls.

3. Eligibility

To use PakPhantom, you must:

  • Be at least 13 years old (or the minimum age required by Discord in your country)
  • Have a valid Discord account in good standing
  • Comply with Discord's Terms of Service
  • Have the necessary permissions to add bots to your server (if adding the bot)

4. Acceptable Use

You agree to use PakPhantom in a manner that is:

  • Respectful — Treat other users and the bot with respect
  • Lawful — Only use the bot for legal purposes
  • Fair — Don't exploit bugs or loopholes in games or economy
  • Reasonable — Don't spam commands or intentionally overload the bot
  • Appropriate — Use features as intended, not to harass others

5. Prohibited Activities

The following activities are strictly prohibited:

Prohibited

Violating any of these rules may result in being blacklisted from using PakPhantom.

  • Exploiting Bugs — Using bugs, glitches, or exploits to gain unfair advantages in games or economy. Report bugs instead.
  • Automation — Using selfbots, macros, scripts, or other automated tools to interact with PakPhantom
  • Spam — Excessive or rapid command usage designed to disrupt the bot or server
  • Harassment — Using PakPhantom features (interactions, marriage, etc.) to harass, stalk, or bully other users
  • Impersonation — Pretending to be a PakPhantom developer or staff member
  • Reverse Engineering — Attempting to decompile, reverse engineer, or extract the bot's source code
  • API Abuse — Directly accessing PakPhantom's APIs or database without authorization
  • Scamming — Using the economy system to scam other users
  • Alt Abuse — Using multiple accounts to farm economy rewards
  • Illegal Content — Using the bot to distribute or promote illegal content

  • Virtual Only — PakPhantom coins have no real-world monetary value
  • No Real Trading — You may not buy, sell, or trade PakPhantom coins for real money, cryptocurrency, or other virtual currencies
  • No Guarantees — We reserve the right to reset, modify, or remove economy data at any time
  • Fair Play — Exploiting the economy system (alt farming, bug exploits, etc.) may result in balance resets or blacklisting
  • Gambling Disclaimer — Casino games (slots, blackjack, roulette) use virtual currency only. No real money is involved
Important

PakPhantom coins are not real currency and cannot be redeemed for anything of real-world value. Casino games are for entertainment purposes only.
