Privacy Policy

How PakPhantom collects, uses, and protects your data. Last updated: January 2024

Quick Summary

PakPhantom collects minimal data necessary for bot functionality. We never store message content, never sell your data, and you can request deletion at any time.

1. Information We Collect

PakPhantom collects only the minimum data required for bot functionality. We categorize this data as follows:

1.1 Automatically Collected Data

  • Discord User ID — Your unique Discord identifier used to track your economy balance, game statistics, and settings
  • Discord Server ID — Used to provide server-specific features and settings
  • Command Usage — Anonymous command usage analytics (which commands are used, not who uses them)

1.2 User-Provided Data

  • Marriage Data — Partner IDs when you use the /marry command
  • Interaction Counts — Number of hugs, kisses, slaps etc. given and received
  • Reminders — Reminder messages and timestamps you set
  • Notes — Personal notes you save using the /note command
  • AFK Messages — AFK status messages you set

1.3 Data We Do NOT Collect

We Never Collect

Message content, voice data, personal information (name, email, age), IP addresses, browsing history, or any data from channels the bot is not invited to.

2. How We Use Your Information

The data we collect is used exclusively for:

  • Bot Functionality — Providing economy balances, game stats, marriage records, reminders, and interaction counters
  • Service Improvement — Anonymous analytics to understand which features are popular and improve the bot
  • Bug Fixing — Error logs (containing no personal data) to identify and fix issues
  • Leaderboards — Displaying server-level rankings and statistics

We never use your data for advertising, marketing, profiling, or selling to third parties.

3. Data Storage & Security

  • Database — Data is stored in MongoDB with encrypted connections
  • Access — Only the bot developer has access to the database
  • Encryption — All database connections use TLS/SSL encryption
  • Backups — Regular backups are maintained for data recovery purposes
  • Retention — Data is retained only as long as necessary for bot functionality

While we take reasonable measures to protect your data, no system is 100% secure. We continuously work to improve our security practices.

4. Data Sharing

We Never Sell Your Data

PakPhantom does not sell, trade, rent, or share your personal data with any third party for commercial purposes. Period.

Data may only be shared in these limited circumstances:

  • Legal Requirements — If required by law or legal process
  • Safety — To prevent harm or illegal activities
  • Discord ToS — Compliance with Discord's Terms of Service

5. Your Rights

You have the following rights regarding your data:

  • Right to Access — Request a copy of all data we store about you
  • Right to Deletion — Request complete deletion of your data
  • Right to Opt-Out — Stop using the bot at any time; remove it from your server
  • Right to Rectification — Request correction of inaccurate data
  • Right to Portability — Request your data in a machine-readable format

6. Data Deletion

To request deletion of your data:

  1. Join our Support Server
  2. Open a support ticket
  3. Request data deletion with your Discord User ID

Data deletion requests are processed within 48 hours. Upon deletion, all your economy data, marriage records, interaction statistics, reminders, and notes will be permanently removed from our database.

When the bot is removed from a server, server-specific data is automatically cleaned up within 30 days.

7. Children's Privacy

PakPhantom is intended for users who meet Discord's minimum age requirements (13 years old, or the minimum age in your country). We do not knowingly collect data from users under these age requirements.

If we become aware that we have collected data from a child under the minimum age, we will promptly delete it. Parents or guardians who believe their child has provided data can contact us for deletion.

8. Third-Party Services

PakPhantom integrates with the following third-party APIs:

  • Discord API — Core bot functionality
  • waifu.pics / nekos.life / purrbot — Interaction GIFs
  • Weather APIs — Weather command data
  • Random animal APIs — Dog, cat, fox images
  • OMDB API — Movie/TV information

These services have their own privacy policies. We only send necessary data (like city names for weather) and never share user IDs or personal information with these APIs.

9. Cookies (Website)

This website uses minimal cookies:

  • Theme Preference — Remembers your selected color theme (localStorage)
  • Cookie Consent — Remembers your cookie banner choice (localStorage)
  • First Visit — Tracks if this is your first visit for welcome animation (localStorage)

We do not use tracking cookies, analytics cookies, or advertising cookies. No third-party tracking scripts are loaded on this website.

10. Changes to This Policy

We may update this Privacy Policy from time to time. When we make significant changes:

  • We will update the "Last Updated" date at the top of this page
  • We may announce changes in our Discord support server
  • Continued use of PakPhantom after changes constitutes acceptance of the updated policy

We encourage you to review this policy periodically.
