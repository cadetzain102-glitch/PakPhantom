Frequently Asked Questions

Quick answers to the most common questions about PakPhantom.

Getting Started

How do I add PakPhantom to my server?

Adding PakPhantom is easy! Follow these steps:

  1. Click the "Add to Server" button on our website
  2. Select your server from the dropdown menu
  3. Grant the required permissions
  4. Click "Authorize"
  5. Type /help in any channel!

That's it! PakPhantom is now ready to use in your server. 🎉

What permissions does PakPhantom need?

PakPhantom requires these permissions to function properly:

  • Send Messages — To respond to commands
  • Embed Links — To send rich embeds
  • Attach Files — For images and GIFs
  • Use External Emojis — For game displays
  • Read Message History — For snipe commands
  • Add Reactions — For polls and games

All permissions are used solely for bot functionality. We never access private channels without being invited.

Is PakPhantom completely free?

Yes! 100% free. PakPhantom is completely free to use with no premium tiers, no paywalls, and no hidden costs. All 94 commands and 22 games are available to everyone.

We believe in making Discord bots accessible to everyone, especially in the Pakistani community. 🇵🇰💚

Commands & Games

How do I use slash commands?

PakPhantom uses Discord's slash commands. Simply type / in any channel and you'll see a list of available commands.

You can also type the beginning of a command name to filter, like /black to find /blackjack.

My game is stuck! What do I do?

If your game session appears stuck, try these steps:

  1. Use the /reset command to clear your stuck game session
  2. Wait 15 minutes — games auto-cleanup after inactivity
  3. Try starting the game again

If the issue persists, report it in our support server.

How do I earn coins?

There are several ways to earn coins:

  • /daily — Claim free coins every 24 hours
  • /work — Work various jobs for coins
  • /blackjack — Win coins by playing blackjack
  • /slots — Try your luck at the slot machine
  • /roulette — Bet on roulette

Tip: Maintain a daily streak for bonus rewards! 🔥

Can I play games with friends?

Yes! Several games support multiplayer mode:

  • /tictactoe — Challenge a friend
  • /connect4 — Play against a friend
  • /rps — Rock Paper Scissors with friends

Just mention the friend you want to challenge and they can accept or decline!

Technical

Why is the bot offline?

If PakPhantom appears offline, it could be due to:

  • Maintenance — Regular updates and improvements
  • Discord Outage — Check Discord Status
  • Hosting Issues — Rare but possible server problems

Check our Status Page for real-time info. The bot usually comes back within minutes.

Slash commands not showing up?

If slash commands aren't appearing, try these fixes:

  1. Make sure the bot has "Use Application Commands" permission
  2. Try restarting your Discord app
  3. Wait a few minutes — Discord can take up to an hour to sync commands
  4. Re-invite the bot with the correct permissions
  5. Check if commands are disabled in channel settings
What data does PakPhantom store?

PakPhantom stores minimal data required for functionality:

  • User ID — For economy balances and game stats
  • Marriage Data — Partner IDs and marriage dates
  • Interaction Counts — Hug/kiss/slap counters
  • Reminders — Your set reminders

We never store message content, personal information, or share data with third parties. See our Privacy Policy for details.

How can I delete my data?

You can request data deletion by:

  1. Joining our Support Server
  2. Opening a ticket in the #support channel
  3. Requesting data deletion with your User ID

We process data deletion requests within 48 hours. All your economy data, marriage records, stats, and reminders will be permanently removed.
